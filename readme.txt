Simple twilio woocommerce order notification system

== Description ==
This plugin lets you add functionality to send twilio notification sms on  a woocommerece new order.

== Installation ==
1. Download the plugin
2. Upload the twilio-woocommerce-order-notification folder to your site's wp-content/plugins
directory
3. Activate the plugin in the WordPress Admin Panel
4. Go to order notification and enter your twilio information
